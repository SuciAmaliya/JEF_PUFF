# JEF_PUFF - Jakarta Electronic Finder

## Deskripsi Project
Jakarta Electronic Finder adalah aplikasi sederhana berbasis Java yang dibuat menggunakan NetBeans. Aplikasi ini bertujuan untuk membantu pengguna mencari tempat servis elektronik di wilayah Jakarta.

Project ini dibuat sebagai final checkpoint untuk tugas akhir mata kuliah Pemrograman Berorientasi Objek.

## Fitur Aplikasi
Beberapa fitur yang sudah dibuat dalam aplikasi ini yaitu:

- Login pengguna
- Register akun baru
- Menu utama aplikasi
- Pencarian tempat servis berdasarkan wilayah Jakarta
- Tampilan daftar toko/tempat servis elektronik
- Detail toko atau tempat servis
- Booking layanan/barang
- Koneksi database MySQL
- Fitur simpan, edit, dan hapus data booking

## Tools yang Digunakan
- Java
- NetBeans
- MySQL
- JDBC / MySQL Connector
- JCalendar
